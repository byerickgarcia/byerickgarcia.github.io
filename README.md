# Erick Garcia — Landing de Conversão
Site estático de alto desempenho orientado a SEO, acessibilidade e conversão.

## Recursos
- SEO completo (OG/Twitter, canonical, JSON-LD)
- Performance (preload, lazy, SW/PWA)
- Acessibilidade (semântica, foco, contraste)
- Conversão (CTA tracking, FAQ, prova social)
- LGPD (consentimento de cookies)
- Estrutura fácil de adaptar

## Como usar
1. Suba a pasta em qualquer host static (Netlify, Vercel, GitHub Pages).
2. Edite telefone do WhatsApp, redes, textos e imagens em `/img`.
3. Cole seus snippets reais de GA4/GTM/Meta Pixel no `index.html`.
4. Ajuste domínio em `robots.txt` e `sitemap.xml`.

## Desenvolvimento
- Edite `styles.css` e `app.js` conforme identidade visual.
- Troque imagens em `/img` mantendo nomes ou atualize referências.
